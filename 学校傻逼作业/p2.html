Componentes del ordenador

Un ordenador es es una máquina electrónica digital programable que ejecuta una serie de comandos para procesar los datos de entrada, obteniendo convenientemente información que posteriormente se envía a las unidades de salida.

Los periféricos o componentes externos de un ordenador son:

Entrada

  1. Placa Base
  2. Procesador
  3. Fuente de alimentación
  4. Memoria RAM
  5. Disco duro
  6. Tarjeta gráfica
  7. Tarjeta de sonido
  8. Ventilador

Salida

  1. Monitor
  2. Impresora
  3. Altavoces
  4. Auriculares
  5. Disco duro
  6. Plotter

Almacenamiento

  1. CD/DVD
  2. Memoria USB
  3. Disco duro externo
